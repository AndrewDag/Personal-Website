Summary

As apart of the Michigan State University Computer Science Capstone experience, I worked on a team of fellow students with RPM, a supply chain and logistics company, to develop an AI Chatbot that would help improve their customer efficiency and reduce the call load for the on-call representitives.

The Chatbot is a backend service that gives dynamic responses to users through the use of OpenAI's ChatGPT API. It offers a variety of different FAQ-style questions that it can answer but it also has the ability to pull information off of Turvo, an API that hosts RPM's .NET C# database which has information in regards to shipments, orders, and more.

Project 2

Summary

Project 2 was the final project of the CSE 335 and had two parts. First, a UML diagram was created for the program. This was done collaboratively in a group.

The second part was the implementation of "Harold Frankenstein's Infernal Laboratory" which was an adapted keyframe animation maker as well as a laboratory with timed elements. These elements of the background would change depending on the time of the animation and there was a power system that allowed for connection between background elements.

This project encompassed new patterns like the Composite, Observer, and Factory, as well as the implementation of a UML design into code.
